# BusBookingFrontEnd

## Description

HUBU 毕业设计-汽车客运在线订票系统前端系统

## Installation

1. git pull 拉取代码
2. 使用前npm i 安装依赖包
3. 运行 npm run serve 启动vue 和 测试服务器（express结合Mock搭建）
4. 管理员登录信息 用户名：admin    密码：123
5. 用户登录信息 用户名：user   密码：123
